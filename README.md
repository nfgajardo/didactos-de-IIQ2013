# Didácticos de IIQ2013

Repositorio base para organizar material docente del curso IIQ2013.

## Estructura

- `01-syllabus/` programa del curso, calendario, bibliografía
- `02-clases/` diapositivas y notas por semana
- `03-guias/` guías de ejercicios
- `04-talleres/` actividades prácticas
- `05-evaluaciones/` controles, pruebas, exámenes
- `06-solucionarios/` soluciones (uso interno)
- `07-recursos/` datasets, scripts, lecturas, material complementario
- `08-actas-y-rubricas/` rúbricas, criterios de evaluación, actas

## Convención sugerida de nombres

`YYYY-MM-DD_tema_version.ext`

Ejemplo: `2026-03-15_balance-materia_v1.pptx`
